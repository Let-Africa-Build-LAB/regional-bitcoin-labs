# Regional Bitcoin Labs

Technical documentation for LAB Regional Bitcoin Labs supporting node experimentation and contributor collaboration.

## Overview

Regional Bitcoin Labs are technical workspaces activated during the LAB pilot year to support in-person experimentation, node operations, and structured open-source collaboration. These labs primarily serve active cohort participants and contributor sprint teams.

## Lab Purpose

- Bitcoin node experimentation
- Lightning testing environments
- Infrastructure prototype refinement
- Contributor sprint collaboration
- Periodic technical workshops and hackathons

## Lab Components

Each lab includes:

- Bitcoin Core node setup
- Lightning Network testing environment
- Dedicated workstations
- Stable internet connectivity
- Power stabilization solutions

## Activation Model

Labs are activated in phases during the Nigeria pilot to validate operational structure before broader continental replication.

## Alignment

Regional Bitcoin Labs exist to support measurable contributor output and infrastructure development within the LAB program.
